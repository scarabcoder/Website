Projects I've Worked On

MyContentDash provides WordPress site owners with a platform containing tools to quickly identify and fix SEO related content issues.

The app uses ReactJS, served from a PHP backend. The administrative UI and API backend exists as a WordPress installation.

MCD also integrates with client WordPress sites using a custom WP plugin, which serves as the bridge for importing content used in audit reports.

My role: I developed the app's frontend on my own, working with another developer to create the backend and handle data management (from BigQuery) that is later displayed on the frontend.

The Blog Fixer is a service that can can repair or make mass updates to post contents quickly.

TBF targets specific common issues and fixes them via a custom WordPress plugin making direct changes to the database, with no impact on site performance and without requiring a plugin to remain active to function.

My role: I joined this project while it was ongoing, starting out with using the plugin on TBF client websites, and later adding new features, such as a tool that utilizes Microsoft Vision and Google image recognition to generate alternative text for image tags for accessibility benefits.

Check it out

CustMap helps companies provide an easy-to-follow roadmap to track the progress of their service or onboarding process with their clients.

Internally, CustMap is a ReactJS app with an ExpressJS backend. It uses MongoDB (via Mongoose) for database storage (MERN stack). For state management, it uses Redux + RTK Query. The app is hosted as a DigitalOcean Ubuntu droplet, using nginx for proxying requests.

My role: I was the sole developer on this project. It was also my first real experience with using ExpressJS and MongoDB, which made it a great learning experience.

MetaGen

MetaGen is a service that can summarize the content of WordPress posts and generates a meta description to be used in SEO plugins.

This is a personal project that is currently in progress. It consists of a MERN (Mongo/Express/React/Node) stack, and utilizes OpenAI's completion AI to create a short summary of post contents.

The project is currently hosted on Heroku.

My role: I am currently the sole developer on this project.
